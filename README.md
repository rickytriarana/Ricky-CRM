# Arana CRM (Offline) — PWA (Gratis, tanpa Play Store)

**v3** memperbaiki bug *modal selalu muncul & tidak bisa ditutup* di sebagian Android.

## Cara Update di GitHub (dari HP)
Upload (replace) file berikut di repo kamu:
- index.html
- app.js
- style.css
- sw.js
- manifest.webmanifest
- icon-192.png
- icon-512.png

## Jika setelah update masih stuck
1) Uninstall PWA (hapus dari home screen)
2) Chrome → Settings → Site settings → Storage → hapus untuk `rickytriarana.github.io`
3) Buka ulang link GitHub Pages → Install lagi
